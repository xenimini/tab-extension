Save Tab - Расширение для Chrome

Простое расширение для сохранения и восстановления URL вкладок.

## Возможности

- 💾 Сохранить URL текущей вкладки
- 🔄 Восстановить URL в новой вкладке  
- 🗑️ Очистить сохраненный URL
- 💾 Автоматическое сохранение (даже после перезапуска Chrome)

Установка

1. Скачайте или склонируйте папку с расширением
2. Откройте Chrome и перейдите по адресу `chrome://extensions/`
3. Включите **"Режим разработчика"** (в правом верхнем углу)
4. Нажмите **"Загрузить распакованное расширение"**
5. Выберите папку с расширением

Как пользоваться

1. Нажмите на иконку расширения в правом верхнем углу Chrome
2. Нажмите **"Save Current Tab"** - URL сохранится
3. Нажмите **"Restore in New Tab"** - откроет сохраненный URL
4. Нажмите **"Clear Saved URL"** - удалит сохранение

- Google Chrome (последняя версия)
- Включенный режим разработчика в расширениях
